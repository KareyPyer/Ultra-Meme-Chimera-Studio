## 🌌 Qu'est-ce que c'est ?

**ULTRA MEME CHIMERA STUDIO** est un environnement de conception générative post-mémétique. Il fusionne la génération de prompts visuels, l'analyse de pièges cognitifs (Basilisk Tracker) et la création d'images chimériques.

L'outil est conçu pour :
- **Explorer l'ontologie du mème** à travers des mutations et hybridations.
- **Détecter et neutraliser** les artefacts épistémiques (biais, autorités, boucles).
- **Forger des images** à l'aide d'un moteur "Chimera Forge".
- **Infecter** des prompts avec des logiques basilisk pour la création d'horreur cognitive.

## ⚙️ Installation

### 1. Prérequis
- Python 3.8 ou supérieur
- Pip (gestionnaire de paquets)

### 2. Cloner le dépôt
```bash
git clone https://github.com/your-username/ultra-meme-chimera.git
cd ultra-meme-chimera
```

### 3. Installation des dépendances
Les dépendances principales sont incluses dans `requirements.txt` (à créer ou via pip).

```bash
pip install -r requirements.txt
```

> **Note**: Si vous ne disposez pas de `requirements.txt`, installez les bibliothèques nécessaires :
> ```bash
> pip install pillow numpy
> ```

### 4. Structure des fichiers
Assurez-vous que les fichiers suivants se trouvent dans le même répertoire que le script principal :
- `MemeChimera.py`
- `meme_prompts_ultra.json`
- `saturation_2026_ultra.json`
- `artifacts_basilisk_extended.json`

## 🚀 Démarrage Rapide

Lancez simplement le script principal :

```bash
python MemeChimera.py
```

### Première utilisation :
1. **Onglet "Fusion / Mono"** : Sélectionnez deux templates pour les fusionner.
2. **Onglet "Analyse + Basilisk"** : Collez un prompt pour analyser ses pièges cognitifs.
3. **Onglet "Chimera Forge"** : Génerez une image via un preset.
4. **Onglet "Mode Infection"** : Combinez template, basilisk et forge.

## 🔮 Fonctionnalités

### 1. **Moteur de Fusion / Mono (Onglet 1)**
- **Fusion** : Combinaison de deux prompts via 4 méthodes (concaténation, entrelacement, hybride, pondéré).
- **Mono** : Génération à partir d'un seul template.
- **Modifications** : Ajout de styles, mutations et hybridations.

### 2. **Mode Chaos (Onglet 2)**
- Niveaux de chaos de 1 à 10.
- Injection automatique d'artefacts Basilisk à partir du niveau 6.
- Génération de prompts extrêmes (réalité glitchée, entités cosmiques).

### 3. **Générateur d'Histoires (Onglet 3)**
- Création de sagas mémétiques en plusieurs chapitres.
- Personnalisation du personnage et du scénario.

### 4. **Analyse & Basilisk Tracker (Onglet 4)**
- Analyse de la complexité des prompts.
- **Scan des artefacts épistémiques** parmi 120+ pièges cognitifs.
- Génération de **contre-prompts** pour neutraliser les basilisks.
- Évaluation du niveau de menace (NUL → CRITIQUE).

### 5. **Batch de Prompts (Onglet 5)**
- Génération de 1 à 50 prompts.
- Exportation en JSON.

### 6. **Chimera Forge (Onglet 6)**
- Génération d'images basées sur des presets visuels.
- **Double-lecture (surface/subtext)** pour un effet basilisk.
- Fusion avec une image source.
- Export du prompt pour des modèles comme Grok.

### 7. **Basilisk Tracker (Onglet 7)**
- Exploration des 120+ artefacts épistémiques.
- Filtrage par catégorie (autorité, ontologie, paradoxe, etc.).
- Affichage des fragments et de leur présence dans les templates.

### 8. **Mode Infection (Onglet 8)**
- **Combinaison ultime** : template mémétique + artefact basilisk + chaos + forge.
- Génération d'un prompt infecté et d'une image chimère.

Bibliothèque d'artefacts épistémiques avec :
- Catégories (autorité, biais, paradoxe, etc.).
- Fragments textuels.
- Niveaux de complexité (1 à 5+).
- Tags.

## 🛡️ Protocole de Sécurité & Éthique

### ⚠️ Avertissement
Ce logiciel contient des **artefacts épistémiques** conçus pour expérimenter la manipulation cognitive. Il n'est pas destiné à un usage malveillant.

### Utilisation Responsable
- **Ne pas utiliser** pour du harcèlement ou de la manipulation non consentie.
- **Environnement de test** : Utilisez-le dans un cadre de recherche ou de création artistique.
- **Auto-évaluation** : L'outil génère des contre-prompts pour vous aider à identifier les pièges.

### Désactivation
Si vous ressentez un inconfort ou une altération de votre confiance ontologique, fermez l'application et lisez le contre-prompt généré dans l'onglet d'analyse.
